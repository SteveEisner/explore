# Explore

Explore is part **wiki**, part **artifact authoring environment**. You gather
documents and data into the wiki, then talk with the built-in agent about
them — by chat or by voice — and together you build **artifacts**:
interactive views (presentations, analysis tools, study aids) that live
right inside the wiki's own pages.

## What it is, simply

- **The wiki is a folder of files.** Markdown pages, data, and artifacts —
  everything in the main panel is one of its files, and the home view lists
  them all.
- **You talk; the agent does.** It reads the wiki, discusses it with you,
  answers from the material, and makes every change itself — you never edit
  a file by hand. Ask, watch the page update live, ask again.
- **Artifacts live inside the pages.** An artifact is an interactive OpenUI
  view embedded in a markdown document: it renders inline where it sits, and
  opens full-screen when you want to work with it.

worklogs — lives in the `design/`, `reports/`, and `worklogs/` folders,
browsable from the home view.
