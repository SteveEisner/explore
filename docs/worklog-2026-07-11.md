New capability: wiki markdown can now embed a live OpenUI application with `<oui-embed src="file.oui"></oui-embed>` (rehype-raw + sanitize-schema allowlist + a component mapping to `GenerativeView`). Verified end-to-end in Chrome against pr-502764-review.oui (tabs interactive inside the embed).
